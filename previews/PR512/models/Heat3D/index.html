Heat PDE

Overview

System type: Affine system
State dimension: 48
Application domain: Mechanical Engineering

Model description

The system is described by the linear differential equations:

\[ \begin{array}{lcl} \dot{x}(t) &=& Ax(t) + Bu(t),\qquad u(t) \in \mathcal{U} \\ y(t) &=& C x(t) \end{array}\]

There are two versions of this benchmark:

  • (time-varying inputs): The inputs can change arbitrarily over time: $\forall t: u(t)\in \mathcal{U}$.

  • (constant inputs): The inputs are uncertain only in their initial value, and constant over time: $u(0)\in \mathcal{U}$, $\dot u (t)= 0$.
